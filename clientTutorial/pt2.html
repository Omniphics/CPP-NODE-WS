C++ Client pt2

Setting Global Variables

Variables

Example
          time_t timer;
          struct tm ltm;
          struct tm y2k = { 0 };
          std::string uri;

These variables is to handle time and the address of the server which is called through the program. Based on your own usage, it can be tweaked but "uri" is important to connect to the server!

Struct

Example
          struct Connection
          {
            int id;
            bool registered = false;
          }m_connection;

Used to store client details sent by the server

          struct pingTimer
          {
            int id;
            bool enabled = true;
            int latestPing = 0;
            int PING_COUNT = 0;
            int interval = 5;
            int PING_LIMIT = 5;
            std::string logFile = "PingPongLog.txt";
          }m_pingTimer;

These are configuration storing ping-pong communcation detail. The configuration can be loaded through a json file or left as default.

Caution: the program does not automatically generate a .txt to store!
