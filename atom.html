Getting Started on Atom

Download & Install

  1. Download and install Atom from https://atom.io/
  2. After running the installer, it will automatically launch and done! Ready to code.

Creating New Project

  1. Create a project folder in an accessible directory. Go to atom "File" → "Open Folder..." → select the directory of project folder, and it will open the folder in atom to be work on, similar to visual studio "Solution Explorer".
  2. Right-click on the directory and select "New File". Name the file and add an extension behind such as .js, .json, .html
  3. Begin writing code!

Extra tip: By installing "platformio-ide-terminal" as additional plugin in Atom, you can run javascript without running command prompt and navigating to the directory.

platformio-ide-terminal

  1. Go to "File" → "Settings" → "Install" → Search "platformio-ide-terminal" → Install
  2. Once installed, look at the botoom left for a "+" symbol and click it to launch.
  3. It should open to your working directory, if not - manually navigate to it.
